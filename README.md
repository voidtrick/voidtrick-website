# Voidtrick - the Website

This is all the code for [voidtrick.com](https://www.voidtrick.com), built with [Astro](https://astro.build/).

This repository is hosted at [Codeberg](https://codeberg.org/voidtrick/voidtrick-website) and then cloned to [Github](https://github.com/voidtrick/voidtrick-website), and then deployed via [Cloudflare Pages](https://pages.cloudflare.com/).
