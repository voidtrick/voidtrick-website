# Voidren- the Website

This is all the code for [voidren.com](https://www.voidren.com), built with [Astro](https://astro.build/).

This repository is hosted at [Codeberg](https://codeberg.org/voidren/website) and then cloned to [Github](https://github.com/voidrenio/website), and then deployed via [Cloudflare Pages](https://pages.cloudflare.com/).
