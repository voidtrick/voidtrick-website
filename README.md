# Voidren- the Website

This is all the code for [voidren.com](https://www.voidren.com), built with [Astro](https://astro.build/).

This repository is hosted at [Codeberg](https://codeberg.org/voidren/website) and cloned to both [Github](https://github.com/voidrenio/website) and my personal [Personal Forgejo instance](https://git.voidren.com/voidren/website), and then deployed via a self-hosted Forgejo worker to a VPS that hosts the website.
